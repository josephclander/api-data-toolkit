# API Data Toolkit

**Utility scripts to clean and manipulate API response data.**

The scripts were created to automate processes and solve problems required in a professional API data handling work.

### Contents

| Function                                                                                      | Description                                                   |
|-----------------------------------------------------------------------------------------------|---------------------------------------------------------------|
| [AddDescriptions](https://github.com/josephclander/api-data-toolkit/tree/main/AddDescriptions)        | Add description fields to connector array of objects that is already formed                    |
| [AddFields](https://github.com/josephclander/api-data-toolkit/tree/main/AddFields)              | Add fields if not present in an object with value 'Undefined' |
| [DifferentObjectKeys](https://github.com/josephclander/api-data-toolkit/tree/main/DifferentObjectKeys)        | Find the key of values different between two objects                   |
| [DisplayOrder](https://github.com/josephclander/api-data-toolkit/tree/main/DisplayOrder)        | Change all values to chronological DisplayOrder               |
| [JsonToCsvConverter](https://github.com/josephclander/api-data-toolkit/tree/main/JsonToCsvConverter)        | Convert an array of objects into a csv format             |
| [ManipulateJson](https://github.com/josephclander/api-data-toolkit/tree/main/ManipulateJson)        | Convert JSON into the Cyclr output format               |
| [ObjectDiffer](https://github.com/josephclander/api-data-toolkit/tree/main/ObjectDiffer)        | Find difference between two objects                    |
| [ScrapeInfoFromHTML](https://github.com/josephclander/api-data-toolkit/tree/main/ScrapeInfoFromHTML)        | Pull key text content from websites when collating API doc info                    |
